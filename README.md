# \## Descripción

# 

# La Bodeguita es una aplicación de escritorio desarrollada en Visual Basic .NET 8 para la gestión de empleados de una cadena de bodegas y tiendas de abarrotes.

# 

# El sistema permite registrar, consultar, modificar y eliminar empleados, clasificándolos en dos tipos: Gerentes y Operarios. Cada tipo de empleado cuenta con características particulares para el cálculo de su pago mensual.

# 

# La aplicación implementa conceptos de Programación Orientada a Objetos como herencia, clases abstractas, encapsulamiento, polimorfismo y manejo de excepciones, además de una interfaz gráfica desarrollada con Windows Forms.

# 

# Funcionalidades

# 

# \* Registro de empleados.

# \* Consulta de empleados registrados.

# \* Modificación de información.

# \* Eliminación de empleados.

# \* Cálculo automático del pago mensual.

# \* Gestión de Gerentes y Operarios.

# \* Validación de datos mediante módulos reutilizables.

# \* Visualización detallada de la información de cada empleado.
